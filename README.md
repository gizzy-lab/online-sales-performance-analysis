# Online Sales Performance Analysis

This project analyzes online sales performance across product categories, regions, and payment methods using Google Sheets.

The objective was to identify the key drivers of revenue and uncover business insights that could support future growth strategies.

## Project Overview

This analysis explores:

- Product category performance
- Regional revenue trends
- Payment method preferences
- Revenue drivers and business opportunities

## Dataset
- Source: Kaggle

## Tools Used
- Google Sheets
- Pivot Table
- Data Cleaning
- Data Visualization
